Directory holding current code of the AlienZoo study line.

Contents:
+ AlienZoov01_start_CON.py 
	- script starting local server that runs control condition 
	  (no computation of CF)
+ AlienZoov01_start_EXP.py
	- script starting local server that runs experimental condition 
	  (with computation of CF)
+ log
	- directory where log files will be written to
+ models
	- directory containing simulated data and models trained
+ static
	- directory containing static game elements (visual assets, images, etc.)
+ templates
	- directory with html code for both control and experimental conditions
